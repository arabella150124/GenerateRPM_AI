# GenerateRPM_AI
Aplikasi ini hanya referensi. Hasil RPM yang dihasilkan oleh AI ini bersifat sebagai referensi awal dan tidak sepenuhnya bersifat final. Bapak/Ibu Guru dapat menyesuaikan kembali isinya dengan kebutuhan, karakteristik peserta didik, serta kondisi pembelajaran di sekolah masing-masing.
